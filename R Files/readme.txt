This file lists the contents of each file and folder in R Files

Extra Saved Models = models saved that were not included in final manuscript

Figures = all Figures included in Manuscipt with previous versions not used in another obsolete folder

obsolete = old files that that are not pertinent for the meta-analysis or are previous copies of data files/scripts

Files:
agg_Z.RDATA = Saved aggression measure mixed effects model
allRnd_Z.RDATA = Saved Random Effects Model
class_Z.RDATA = Saved class mixed effects model
Effect Size Calculations.R = R code for effect size calculations
MCMCglmm_Z.R = R code for MCMCglmm and figures
rcond_Z.RDATA = Saved random effects model that has uncontrolled condition removed
rcondcolor_Z.RDATA = Saved class mixed effects model that has uncontrolled condition removed
rsandc_Z.RDATA = Saved random effects model that has uncontrolled social rank and uncontrolled condition removed
rsandcolor_Z.RDATA = Saved class mixed effects model that has uncontrolled social rank and uncontrolled condition removed
rsocial_Z.RDATA = Saved random effects model that has uncontrolled social rank removed
rsocolor_Z.RDATA = Saved class mixed effects model has uncontrolled social rank removed
